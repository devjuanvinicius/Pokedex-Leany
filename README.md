# Pokédex

Pokédex web construída como desafio técnico para uma vaga de frontend React. Consome a [PokeAPI](https://pokeapi.co) pública.

Ver [CLAUDE.md](./CLAUDE.md) para stack, convenções de código e decisões de arquitetura, e [ROADMAP.md](./ROADMAP.md) para o progresso do projeto.

## Rodando o projeto

```bash
npm install
npm run dev      # ambiente de desenvolvimento
npm run build    # build de produção
npm run preview  # pré-visualizar o build
npm run lint     # ESLint
npm run test     # Vitest
```
